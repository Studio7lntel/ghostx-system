🔄 Roblox Experience Ban — Sync Blacklist
💡 ใช้ Roblox Open Cloud API เพื่อ ban/unban ใน Experience โดยตรง — ผู้เล่นจะไม่สามารถ rejoin ได้แม้ไม่อยู่ใน server
📌 หา Universe ID: ไปที่ create.roblox.com → เลือก Experience → ดูตัวเลขใน URL ช่อง /experiences/ID/overview
🔑 สร้าง API Key: create.roblox.com/credentials → สร้าง API Key → เลือก System: Experience Bans → Operations: Read, Write
📄 SyncBan — วางใน ServerScriptService (ทำงานฝั่ง Server เท่านั้น)
⚠️ Script นี้รันบน Server Script เท่านั้น — ไม่ต้องวางใน LocalScript
⚠️ Open Cloud API Key ห้ามเปิดเผย — เก็บไว้ใน Script ฝั่ง Server เท่านั้น
📄 SyncBan Webhook — ใช้ใน api.php (ตัวเลือกเพิ่มเติม)
📌 เพิ่มโค้ดนี้ใน api.php ในฟังก์ชัน ban/unban เพื่อให้ระบบ sync ไปยัง Roblox อัตโนมัติทุกครั้งที่แบนจากเว็บ
